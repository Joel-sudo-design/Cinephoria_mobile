# Cinephoria mobile

Application mobile **Flutter** pour le site Cinéphoria.

## Description

Cinephoria Mobile est une application développée avec **Flutter**, permettant aux utilisateurs d’accéder à leurs **commandes** et de les **visualiser sous forme de QR code** directement depuis leur smartphone.

Elle est conçue comme le prolongement mobile du site Cinéphoria, avec une interface simple, rapide et orientée utilisateur.

## Fonctionnalités

- Authentification utilisateur
- Consultation des commandes
- Affichage d’un **QR code associé à chaque commande**
- Interface mobile fluide et responsive (Flutter)
